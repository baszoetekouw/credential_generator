SURFconext passphrase generator

Introduction

This page offers a safe way to exchange passphrases with the SURFconext Operations Team. By pressing the button below, a new passphrase will be generated for you. Please write it down immediately, as it is given out only once.

The reference code uniquely corresponds to the passphrase, but it is impossible generate one from the other. Therefore, the reference code can be safely relayed to the SURFconext Operations Team over an insecure communications channel (phone, email, etc), who can look up the corresponding passphrase.
